# 第二章\~数组

## 2.1 数组的声明、创建和初始化

```java
// 一维数组的声明
<数据类型>[] <变量名>; // or Java 不推荐的格式(后面不再单独列出):<数据类型> <变量名>[];
// eg:
int[] array1;
// 一维数组的创建
new <数据类型>[<数组长度>];
// eg:
array1 = new int[5];
array1 = new int[]{1, 2, 3, 4, 5};

// 二维数组的声明
<数据类型>[][] <变量名>;
// eg:
int[][] array2;
// 二维数组的初建
new <数据类型>[<数组长度>][<数组长度>];
// eg:
array2 = new int[3][4];
array2 = new int[][] {
    new int[4],
    {1, 2, 3, 4}
};

// 当声明和创建在同一行时，数组的初始化可以省略 new <数据类型><维度>
// eg:
int[] arr1 = {1, 2, 3, 4, 5};
int[][] arr2 = {
    {1, 2, 3, 4, 5},
    new int[2]
};


int[][] arr2;
arr2 = {
        {1, 2, 3, 4, 5},
        {6, 7, 8, 9, 0}
};
```

## 2.2 数组元素的使用

```java
// 访问索引对应的位置的元素值
int array = new int[4];
System.out.println(array[2]); // 输出 0，整形的默认值是 0
// 当访问的索引超出数组的范围时，将排除 ArrayIndexOutOfBoundsException
// 抛出异常包括小于 0 和大于等于数组长度
```
